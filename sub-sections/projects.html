About This Project

Sensei-Bryan was born from the frustration of AI tools that simply provide answers without encouraging deeper learning. Traditional AI assistants often create a passive learning experience where users become dependent on immediate solutions. This project flips that paradigm by implementing a teaching methodology that emphasizes understanding over memorization.

The application uses advanced prompt engineering with OpenRouter's API to create dynamic learning experiences. Rather than giving direct answers, Sensei-Bryan guides users through problem-solving processes, asks probing questions, and provides scaffolded learning that builds genuine comprehension. The interactive flashcard system uses spaced repetition algorithms to optimize retention, while the practice test feature provides detailed analytics to identify knowledge gaps.

Built with a focus on accessibility and user experience, the interface adapts to different learning styles and maintains engagement through gamification elements. The project demonstrates how AI can be leveraged not just to provide information, but to fundamentally improve how we learn and retain knowledge.

About This Project

This trading bot represents a deep dive into the intersection of machine learning and financial markets. The project emerged from a fascination with whether neural networks could identify patterns in stock price movements that traditional analysis might miss. Using PyTorch, I implemented a sophisticated GRU-based architecture specifically designed for sequential data prediction.

The system goes beyond simple price prediction by incorporating multiple technical indicators (SMA, RSI, MACD) to create a rich feature space for the neural network. Each GRU layer with 128 hidden units captures different temporal dependencies in the data, while the SiLU activation function helps with gradient flow during training. The bot uses Mean Squared Error loss and implements configurable trading strategies based on prediction confidence.

While this is an educational project exploring ML applications in finance, it demonstrates key concepts in time series forecasting, data preprocessing, and automated decision-making systems. The codebase emphasizes modularity and configurability, making it easy to experiment with different architectures and trading parameters.
